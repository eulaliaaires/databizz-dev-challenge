# databizz-dev-challenge
Repositório para o desafio dev da Databizz


### Orientações para executar o projeto

>> #### Passo 1: Clonar o repositório do projeto
>>> ``` git clone https://github.com/eulaliaaires/databizz-dev-challenge.git ```

 >> #### Passo 2: Acessar a raiz do projeto
>>> ``` cd databizz-dev-challenge ```  
 >>> ``` cd cd databizz-dev-challenge ```

>> #### Passo 3: Instalar dependências
 >>> ``` npm install ```

>> #### Passo 4: Executar o projeto 
 >>> ``` ng serve```
 
>> #### Passo 5: Visualizar o projeto 
>>> Acesse ``` http://localhost:4200```
